def categorise(feature,bins,bin_names):
    return pd.cut(movies[feature],bins,labels=bin_names)
In [66]:

#Define a function to calculate the profit series
def Profit(feature1,feature2):
    return movies[feature1]-movies[feature2]
In [69]:

profit=movies.copy()

Conclusions¶

So,
Popular genres w.r.t decades were:
1960-1970 - Science Fiction|Mystery|Adventure
1970-1980 - Adventure|Action|Science Fiction
1980-1990 - Adventure|Comedy|Science Fiction|Family
1990-2000 - Drama
2000-2010 - Action|Adventure|Fantasy|Science Fiction
2010-2015 - Action|Adventure|Science Fiction|Thriller

The top 5 films with the highest profit are Avatar(2544505847), Star Wars: The Force Awakens, Titanic, Jurassic World,
Furious 7(1316249360).

The most liked runtime was around 140-200 minutes.

The number of movies released every year is increasing drastically.
